画面に表示されている数字を順にできるだけ早くクリックするゲームです。
難易度を選択すると、3カウントの後に数字が表示されます。

This game is to click the numbers displayed on the screen in order as quickly as possible.
When the difficulty level is selected, the numbers appear after the three counts.

数字探しゲーム 1/7 Number Search Game 1/7

＜かんたん＞
1~10の数字が出題されます。
目標タイムは8.0秒です。

「Easy」
Numbers from 1~10 are given.
Target time is 8.0sec.

数字探しゲーム 2/7 Number Search Game 2/7

＜ふつう＞
1~20の数字が出題されます。
目標タイムは18.0秒です。

「Normal」
Numbers from 1~20 are given.
Target time is 18.0sec.

数字探しゲーム 3/7 Number Search Game 3/7

＜むずかしい＞
1~30の数字が出題されます。
目標タイムは28.0秒です。
※難易度むずかしいではスクロールしないと一部の数字が見えません。

「Hard」
Numbers from 1~30 are given.
Target time is 28.0sec.
※In the hard difficulty level, you have to scroll to see some of the numbers.

数字探しゲーム 4/7 Number Search Game 4/7

画面上の数字を直接クリックしてください。
正しい順番であれば色が変わります。そうでなければ色は変わりません。

Click directly on the numbers on the screen.
If they are in the correct order, they will change color. If not, the color will not change.

数字探しゲーム 5/7 Number Search Game 5/7

全ての数字の色が変わればクリアです。
Timeをもとに, Rank (S>A>B>C>D)が決定されます。
目標タイムを上回ればSランクです！

The game is cleared when all numbers change color.
Rank (S>A>B>C>D) is determined based on Time.
If the target time is exceeded, it is Rank S!

数字探しゲーム 6/7 Number Search Game 6/7

目標タイムより早く数字を見つけられるように頑張りましょう！

Do your best to find the numbers faster than target time!
